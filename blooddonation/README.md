# NSS Blood Donation System

A Django-based web application to manage blood donation requests and donors.

## Features
- Donor registration with OTP verification
- Blood request system
- Admin dashboard
- Donor matching
- Blood camp module

## Tech Stack
- Django
- SQLite
- Tailwind CSS

## How to Run
1. Clone repo
2. Create virtual environment
3. Install requirements
4. Run server

## Future Improvements
- Real OTP integration
- Notification system
- Smart donor matching
